## 🚀 About Me

> **Data Engineer** with **1+ year** of hands-on experience building end-to-end data solutions for top eCommerce platforms — **Shopee, TikTok Shop & Amazon**. I handle the full data stack: from raw ingestion to warehouse design to business dashboards — and leverage AI to automate what used to take hours.

- 🏗️ Designed & maintained **data pipelines** processing **tens of thousands of records/day** across eCommerce platforms
- ⚡ Implemented **batch & real-time streaming** architectures using **Kafka + Debezium**
- 🏆 Built **Medallion Architecture** (Bronze → Silver → Gold) data warehouses on **GCP & AWS**
- 🤖 Automated **Amazon Ads optimization** using **Claude AI (Claude Code)** — saving **10s of hours/week**
- 📊 Delivered **custom dashboards** tailored to business KPIs (Ads, Orders, Inventory, Returns, Traffic)
- 🌐 Full-stack Data capability: **DE + DA** for end-to-end Amazon sellers
